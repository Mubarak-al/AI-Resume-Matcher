# 🚀 AI Resume Matcher (RAG + LLM)

🔗 **Live Demo:** "https://ai-resume-matcher-tedrray59bwgafvsameyoy.streamlit.app/"

---

## 📌 Overview

AI-powered Resume Matcher that compares resumes with job descriptions using **RAG (Retrieval-Augmented Generation)**.

---

## ⚙️ Features

* 📄 Upload Resume (PDF / DOCX / TXT)
* 🔍 Semantic Job Matching using FAISS
* 🤖 LLM-based Evaluation (Match Score, Missing Skills, Improvements)
* 📊 Visual Similarity Scores
* 📜 Analysis History
